# 🪐 LinkSync

A private, real-time link sharing app with a space theme. Add links on one device and they instantly appear on all others in the same room — no account, no install, just open the file in a browser.

![Space themed interface with floating planet links on a starfield background]

---

## Features

- **Private rooms** — each room has a random 6-character code. Only people with the code can join, nothing is publicly listed
- **Real-time sync** — links sync across all devices in the same room
- **Room history** — the setup screen shows all your recent rooms as large cards so you can rejoin with one click
- **Custom room names & colours** — rename any room and pick a colour for its planet, saved to your browser
- **No account required** — no sign-up, no email, no personal data stored anywhere
- **Single file** — the entire app is one HTML file you can open locally or host anywhere

---

## How to use

| | |
|---|---|
| **Create** | Click "Create room" to generate a private 6-character code |
| **Share** | Send the code to anyone you want in the room |
| **Join** | Enter the code on any device to sync instantly |
| **Privacy** | Rooms are private by code only — no list of rooms is ever exposed |
| **History** | Your recently joined rooms are saved in the browser for quick access |

## Tech stack

| | |
|---|---|
| **Frontend** | Vanilla HTML, CSS, JavaScript — no frameworks |
| **Backend** | Firebase Realtime Database (free tier) |
| **Fonts** | Orbitron, Share Tech Mono, Exo 2 via Google Fonts |
| **Hosting** | Any static host — GitHub Pages, Netlify, Vercel, or just a local file |
